# zshsetup
This contains zsh setup with plugins that makes it easier for you like syntax-hightlighting and auto-suggestion
# Download the file and apply directly
- download with curl ```curl -fsSL https://raw.githubusercontent.com/justinmihigo/zshsetup/main/zshsetup.sh | sh -```
- or clone the repo change directory  then ```chmod +x zshsetup.sh``` then run with ```./zshsetup.sh```
# Post installation
run command ```exec zsh```
